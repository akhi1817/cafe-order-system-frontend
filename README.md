## 📖 Project Overview

Cafe Order Management System is a full-stack MERN application designed to simplify cafe operations. The system allows customers to browse menu items, add products to their cart, place orders, and enables administrators to manage menu items and customer orders through an admin dashboard.

This project was developed to demonstrate full-stack web development skills using React.js, Node.js, Express.js, and MongoDB.

---

## 🚀 Features

### Customer Features

* View available cafe menu items
* Add products to cart

## ⚙️ Installation

### Clone Repository

```bash
git clone <repository-url>
```

### Navigate to Project Folder

```bash
cd cafe-order-management-system
```

### Install Dependencies

```bash
npm install
```

### Run Frontend

```bash
npm start
```

### Backend Setup

Configure the backend server and MongoDB connection separately using environment variables.

Example:

```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

## 📸 Project Workflow

1. Customer visits the application.
2. Customer browses available menu items.
3. Customer adds items to the cart.
4. Cart automatically updates quantities and total amount.
5. Customer places an order.
6. Admin logs into the dashboard.
7. Admin manages menu items and customer orders.
